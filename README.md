This is a simple MC Mod that adds Rubies into the game!

This is a resource material that can be found and mined just like other ores.

Ruby is not only a very strong material, but it has hidden benefits.

-------------------------------
In this .jar holds:

Ruby Armor Set,
Ruby Tool Set,
Ruby Ore Block,
Ruby Block,
Ruby Recipes,
Ruby
